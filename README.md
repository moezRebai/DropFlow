🪣 DropFlow — Smart Furniture Delivery Management

DropFlow is a modern furniture delivery management system designed to simplify logistics operations for delivery businesses. It centralizes client management, delivery routes, invoices, and scheduling in a clean and intuitive interface.

🚀 Key Features

Client Management – Track customers, delivery addresses, and order history.

Delivery Planning – Create and optimize delivery routes with assigned drivers and trucks.

Invoices & Reports – Generate invoices, delivery sheets, and route summaries automatically.

Monitoring Dashboard – Follow delivery progress and performance in real time.

Modern UI/UX – Built with a responsive, elegant WPF interface following the MVVM pattern.

🧩 Tech Stack

Frontend (Desktop): WPF (.NET 8), MVVM Toolkit, MaterialDesignInXaml, Maehapass Theme

Backend: ASP.NET Core Web API

Database: SQL Server with Entity Framework Core

Architecture: Dependency Injection, MVVM, and RESTful communication

💡 Highlights

Dependency-injected ViewModels for testability and maintainability.

Consistent design system with centralized theming and reusable styles.

Scalable architecture ready for multi-branch or multi-user environments.
